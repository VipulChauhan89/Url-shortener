## About

This URL Shortener is a web application that allows users to shorten long URLs into more concise, shareable links. It's built using several technologies, including React, Vite, Tailwind CSS, and an external URL shortening API. The primary goal is to provide a user-friendly and responsive platform for URL shortening.

## Features

- Shorten long URLs into concise, shareable links.

## Getting Started

To get started with the URL Shortener App, follow these steps after cloning the repository:

### Prerequisites

Make sure you have the following software installed:

- Node.js
- npm or yarn

### Installation

1. Clone this repository:

    ```sh
    git clone https://github.com/VipulChauhan89/Url-shortener.git
    ```

2. Navigate to the project directory:
    ```sh
    cd Url-shortener
    ```
3. Install dependencies:
    ```sh
    npm install
    ```
4. Start the development server:
    ```sh
    npm run dev
    ```
5. Open your browser and access the app at http://localhost:3000.

## Usage

- Enter a long URL, click the "Shorten" button, and you will receive a shortened URL.
- Click the "Copy" button to copy the shortened URL to your clipboard.
- Reset buton to repeat the process further to shorten more urls.
